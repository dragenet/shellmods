# alias-yarn

Shorthand aliases for common Yarn commands.

## Aliases

| Alias | Command                          |
|-------|----------------------------------|
| `y`   | `yarn`                           |
| `yi`  | `yarn install`                   |
| `ya`  | `yarn add`                       |
| `yr`  | `yarn run`                       |
| `ylp` | `yarn lint --fix && yarn prettier` |

## Prerequisites

- Yarn installed and available in `$PATH`.
