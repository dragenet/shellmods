# bun

Sets up the Bun JavaScript runtime environment.

## What It Does

- Loads Bun shell completions (if available).
- Exports `BUN_INSTALL` pointing to `~/.bun`.
- Adds the Bun binary directory to `$PATH`.

## Prerequisites

- Bun installed at `~/.bun`. Install via `curl -fsSL https://bun.sh/install | bash`.
