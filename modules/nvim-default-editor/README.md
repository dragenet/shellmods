# nvim-default-editor

Sets `nvim` as the default text editor.

## What It Does

- Exports `EDITOR=nvim` for both local and remote (SSH) sessions.
- Exports `VISUAL=nvim` for both local and remote (SSH) sessions.

## Notes

Mutually exclusive with `nano-default-editor` and `vim-default-editor`. Enable only one editor module at a time.
